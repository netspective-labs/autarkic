# Web UX Design System and Layouts

## Layout patterns (most common)

- App shell (header + sidebar + content)
- Sidebar layout (fixed, collapsible, icon-only, overlay)
- Top nav layout (header tabs with optional secondary nav)
- Split view (list-detail, master-detail)
- Three-pane layout (nav + list + detail)
- Dashboard layout (widgets, cards, grid)
- Workspace layout (toolbar + canvas + inspector/right rail)
- Full-page editor (header actions + form sections)
- Table-centric layout (filters + table + detail drawer)
- Wizard layout (stepper + staged content)
- Search-first layout (search + results + refiners)
- Kanban layout (columns + cards)
- Timeline layout (events + filters)
- Settings layout (section nav + forms)
- Auth layout (centered card, minimal chrome)
- Landing/marketing shell (hero + sections + footer)

## Global structural regions

- Top app bar (header)
